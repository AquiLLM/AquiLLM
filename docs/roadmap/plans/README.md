# Plan Archive

This folder is organized by plan status.

## Buckets

- `active/`: in-flight or partially completed workstreams.
- `pending/`: approved/planned work that has not started.
- `completed/`: implemented plan artifacts kept as historical execution records.
- `superseded/`: replaced plans kept for context and traceability.
